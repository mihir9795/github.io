Magnetic Particle Inspection - Computer Vision

  • Tech Stack: Python, Camera, Raspberry Pi, OpenCV, Machine Learning, Tensorflow Lite

1. Magnetic partical inspection using computer vision. (crack detection), NDT, CV

2. Quality is a key role in any inspection process.

3. Computer-vision based magnetic particle inspection (MPICV) is accurate, precise, reliable and is designed for the inspection of ferrous components up to 600 mm in length.

4. This self-contained system with the camera sensor, edge device, UV light source is a low-cost unit that detects and estimate the length of surface cracks using the wet continuous method of MPI.

5. This magnetic particle inspection with CV would be a reinforcement learning-based system that will continuously learn and make itself accurate to detect cracks and length of cracks with the help of AI/ML algorithms with precision and accuracy.
